# 📘 GenAI & Python Training – Session Plan

## 🧾 Overview

This repository contains the complete session plan for a **GenAI + Python Training Program**. The course is designed to cover fundamentals of Python, Natural Language Processing (NLP), Generative AI tools, and building real-world AI applications using Streamlit and APIs.

---

## 🗓️ Course Timeline

**Start Date:** 6 April 2026
**End Date:** 9 May 2026
**Duration:** ~1 Month
**Daily Commitment:** 2 Hours

---

* Introduction to Python

* Hands-on Python coding
