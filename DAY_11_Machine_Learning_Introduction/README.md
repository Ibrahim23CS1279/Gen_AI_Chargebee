# What is Machine Learning?

Machine Learning is a branch of Artificial Intelligence (AI) that allows systems to learn from data and improve performance without being explicitly programmed.

Machine Learning is used in:
- Recommendation systems
- Chatbots
- Fraud detection
- Image recognition
- Healthcare applications
- Finance systems
- Self-driving cars

---

# Types of Machine Learning

## 1. Supervised Learning

Supervised Learning uses labeled data for training.

### Examples
- House price prediction
- Email spam detection
- Student marks prediction

### Common Algorithms
- Linear Regression
- Logistic Regression
- Decision Trees
- Random Forest

---

## 2. Unsupervised Learning

Unsupervised Learning works with unlabeled data and finds hidden patterns.

### Examples
- Customer segmentation
- Recommendation systems
- Clustering

### Common Algorithms
- K-Means Clustering
- Hierarchical Clustering
- PCA
