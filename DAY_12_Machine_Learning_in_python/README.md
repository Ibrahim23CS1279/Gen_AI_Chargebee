# Machine Learning Introduction

Machine Learning (ML) is a branch of Artificial Intelligence that allows systems to learn patterns from data and make predictions or decisions.

## Types of Machine Learning

### 1. Supervised Learning
The model learns using labeled data.

Examples:
- Linear Regression
- Logistic Regression
- Decision Trees

### 2. Unsupervised Learning
The model learns patterns from unlabeled data.

Examples:
- Clustering
- Dimensionality Reduction
