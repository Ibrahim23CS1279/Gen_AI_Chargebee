# 🐍 Day 2: Functions, OOPs & Modules

This session focuses on writing reusable code using functions, understanding object-oriented programming, and organizing code using modules.

---

## 📌 Topics Covered

### 1. Functions

Functions help avoid repetition and make code reusable.

#### Example:

```python
def abc():
    print("hello world")
```

* Functions can take inputs (parameters)
* Functions can return values

---

### 2. Object-Oriented Programming (OOP)

Python supports OOP concepts such as:

* Classes
* Objects
* Inheritance

---

### 3. Classes & Objects

A class is a blueprint, and objects are instances of that class.

#### Example:

```python
class First:
    a = 78

    def method1(self):
        print("Method1")
```

---

### 4. Inheritance

Inheritance allows one class to reuse properties and methods of another.

#### Example:

```python
class Second(First):
    b = 45
```

---

### 5. Modules

A module is a Python file that contains functions or variables.

We can import modules in different ways:

* `import module`
* `import module as alias`
* `from module import function`
* `from module import *`

---

## 🎯 Learning Outcomes

By the end of this session, you should be able to:

* Write reusable functions
* Create and use classes and objects
* Understand inheritance
* Organize code using modules
* Import functions in different ways
