# 🐍 Day 1: Basic Python Fundamentals

This session covers the core building blocks of Python programming, including data types, collections, string operations, and basic control flow.

---

## 📌 Topics Covered

### 1. Comments in Python

* Single-line comments using `#`
* Multi-line comments using triple quotes (`""" """`)

---

### 2. Fundamental Data Types

* `int` → Integer values
* `float` → Decimal numbers
* `str` → Strings (text)
* `complex` → Complex numbers
* `bool` → Boolean values (`True` / `False`)

---

### 3. Collection Data Types

* `list` → Ordered, mutable collection
* `tuple` → Ordered, immutable collection
* `set` → Unordered, unique elements
* `dict` → Key-value pairs

---

### 4. String Operations

* Indexing (positive & negative)
* Slicing
* Step slicing
* Reversing strings
* Strip functions:

  * `strip()`
  * `lstrip()`
  * `rstrip()`

---

### 5. Mutability

* Lists are mutable (can be changed)
* Tuples are immutable (cannot be changed)

---

### 6. Dictionary Usage

* Access values using keys
* Nested dictionaries

---

### 7. Conditional Statements & Loops

* `if`, `elif`, `else`
* `while` loop
* Example: Password validation system with retry limit

---

## 🎯 Learning Outcomes

By the end of this session, you should be able to:

* Understand Python data types
* Work with collections (list, tuple, set, dict)
* Perform string operations
* Write basic conditional logic
* Use loops for repeated execution
